🔑 Autenticação Google
Faça upload do arquivo JSON da sua Service Account para autenticar com o Google.
✅
📋 Como configurar (mesmo processo do CQS)
  1. Acesse console.cloud.google.com
  2. Crie ou selecione um projeto → APIs & Services → Enable APIs
  3. Habilite: Google Docs API, Google Drive API e Google Sheets API
  4. Crie uma Service Account → gere uma chave JSON → faça upload acima
  5. Copie o e-mail da Service Account (ex: nome@projeto.iam.gserviceaccount.com)
  6. Em cada Google Doc: compartilhe com esse e-mail como Comentarista
  7. Na planilha de configurações: compartilhe com esse e-mail como Editor
📚 Disciplinas
Cadastre cada disciplina com o nome completo, a sigla (código) e o ID do Google Doc.
O ID fica na URL: docs.google.com/document/d/[ID AQUI]/edit
☁️ Sincronizar com Google Sheets
Mantenha as disciplinas espelhadas em uma planilha Google Sheets. A aba disciplinas deve ter cabeçalho: Nome | Sigla | DocID.
📦 Cache de Conteúdo
O conteúdo das abas é armazenado localmente no navegador (IndexedDB) para abertura instantânea no viewer. Sincronizar atualiza apenas as abas que tiveram alterações desde o último sync.
